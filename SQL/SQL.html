SQL

SQL (Structured Query Language) — это язык запросов, используемый для взаимодействия с базами данных. Он позволяет выполнять различные операции, такие как извлечение данных, вставка новых записей, обновление существующих и удаление данных. SQL является стандартом для большинства систем управления базами данных, самые известные из них MySQL, PostgreSQL, SQLite.
