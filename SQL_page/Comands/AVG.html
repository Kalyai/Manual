AVG в SQL

Оператор AVG в языке SQL применяется в связке с SELECT и используется для вычисления среднего значения числового столбца в таблице

Примеры:
  • Вычисление среднего значения 
    SELECT AVG(Age) FROM employees

    • В выводе будет средний возвраст всех `employees`

Используемые команды:

  • SELECT - Выбрать столбцы из таблицы
  • AVG - Сума чисел в столбце поделить на количество строк в столбце
  • FROM name - Из таблицы/базы данных с именем name
