COUNT в SQL

Оператор COUNT в SQL нужен для подсчета строк, которые были бы выведены в результате запроса. То есть оператор COUNT выводит количество строк, которые соответствовали заданым условиям.

Примеры:
  • COUNT с условием 
    SELECT COUNT(*) FROM employees
WHERE salary > 100000

    • В выводе увидим число, которое соответствует количеству сотрудников с зарплатой выше 100к

Используемые команды:

  • SELECT - Выбрать столбцы из таблицы (знак `*` значит все столбцы)
  • COUNT - Подсчитывает количество строк
  • FROM name - Из таблицы/базы данных с именем name
  • WHERE - Оператор для добавления условий
