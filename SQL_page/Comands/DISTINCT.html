DISTINCT в SQL

Оператор DISTINCT в языке SQL используется для удаления дубликатов из результирующего набора данных. Возвращает только уникальные столбцы.

Примеры:
  • Выбрать только уникальные значения из таблицы с измерением температуры 
    SELECT DISTINCT Temperature FROM Measurements

    • Выбираем только уникальные значения температуры в таблице с измерениями

Используемые команды:

  • SELECT - Выбрать что-то, откуда-то
  • DISTINC - Выбрать только уникальное
  • FROM name - Из таблицы/базы данных с именем name
