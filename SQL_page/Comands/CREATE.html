CREATE в SQL

Оператор CREATE в языке SQL используется для создания новых баз данных, таблиц, индексов, представлений и других объектов базы данных. У создаваемого объекта можно настроить структуру и характеристики.

Примеры:
  • Создание базы данных 
    CREATE DATABASE employees

    • Создаем базу данных `employees`

  • Создаем таблицу 
    CREATE TABLE info(
    Id INTEGER PRIMARY KEY AUTO_INCREMENT,
    Name VARCHAR(255) NOT NULL
)

    • Создаем таблицу `info` со столбцами Id, Name. Также задали им какие-то параметры

  • Создание уникальный индекс 
    CREATE UNIQUE INDEX
idx_unique_name ON info(name)

    • Создаем уникальный индекс в столбцу name

Используемые команды:

  • CREATE - Создать что-либо
  • DATABASE name - База данных с именем name
  • TABLE - Таблицы с именем name
  • ON - Используется для установления соединения
