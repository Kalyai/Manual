DROP в SQL

Оператор DROP в языке SQL используется для удаления объектов базы данных, таких как таблицы или индексы. Этот оператор не только удаляет данные внутри объекта, но и сам объект.

Примеры:
  • Удаление базы даннных 
    DROP DATABASE base

    • Удаляем базу данных с именем base

  • Удаление таблицы 
    DROP TABLE employees FROM base

    • Удаляем таблицу employees из базы данных base

  • Удаления индекса 
    DROP INDEX FOREIGN KEY ON employees

    • Удаляем индекс внешний ключ из таблицы employees

Используемые команды:

  • SELECT - Выбрать что-то, откуда-то
  • DISTINC - Выбрать только уникальное
  • FROM name - Из таблицы/базы данных с именем name
