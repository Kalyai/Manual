AS в SQL

Оператор AS в языке SQL используется для присвоения псевдонима столбцу или таблице, чтобы упростить работу с ними или сделать вывод более читаемым. Псевдонимы позволяют временно использовать другое имя в рамках конкретного запроса.

Примеры:
  • Псевдоним для объединения столбцов 
    SELECT First_name + ' ' + Second_name
AS Full_name FROM employees

    • Мы выбираем столбцы с именем и фамилией из `employees`

    Дальше в выводе объединяем их в один столбец `Full_name` с пробелом между именем и фамлиией

  • Псевдоним для более удобства в запросе 
    SELECT First_name + ' ' + Last_name
AS Full_name FROM employees
WHERE Full_name LIKE 'Kalyaev Sergei'

    Мы также выбираем столбцы с именем и фамилией из `employees`

    Дальше объединяем их в один столбец `Full_name` с пробелом между именем и фамлиией

    Потом сравниваем полное имя со строчкой Kalyaev Sergei

    Удобство заключается в том, что мы сразу сравниваем полное имя, а не по отдельности имя и фамилию

Используемые команды:

  • SELECT - Выбрать столбцы из таблицы/базы данных (знак `*` означает выбрать все)
  • AS - Назвать как `name`, создает псевдоним, также может объединять столбцы
  • FROM name - Из таблицы/базы данных с именем name
  • WHERE - Оператор для добавления условий
  • LIKE - Сравнение, Это как Это -> 'Ivanov Ivan' LIKE 'Ivanov Ivan'
