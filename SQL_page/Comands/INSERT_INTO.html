INSERT INTO в SQL

Оператор INSERT в языке SQL используется для добавления новых записей в таблицу базы данных. В связке с VALUES, Он позволяет указать значения для каждого столбца при вставке новой строки данных.

Примеры:
  • Вставка значений в таблицу 
    INSERT INTO employees(First_name, Second_name, Age)
VALUES('Sergei', 'Kalyaev', 18)

    • Вставляем значения в таблицу `employees` и записоваем их в `First_name`, `Second_name`, `age`

Используемые команды:

  • INSERT INTO - Вставка значений
  • VALUES - сами значения для вставка
