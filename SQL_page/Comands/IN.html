IN в SQL

Оператор IN в языке SQL используется для фильтрации данных по нескольким значениям в столбце. Он позволяет создать условие, включающее строки, значения в которых совпадают с любым из указанных в списке значений.

Примеры:
  • Фильтрация при помощи IN 
    SELECT First_name, Second_name FROM Employees
WHERE Post IN ('Developer')

    • Выбираем имя сотрудника из таблицы `Employees`

    Фильтруем сотрудников по должности `Developer`

Используемые команды:

  • SELECT - Выбрать что-то, откуда-то
  • FROM name - Из таблицы/базы данных с именем name
  • IN - Совпадает с чем-либо из (...)
