FROM в SQL

Оператор FROM в языке SQL используется для указания таблицы или набора таблиц, из которых производится выборка данных. Он предоставляет контекст, в котором оператор SELECT извлекает данные.

Примеры:
  • Выборка строк из таблицы Employees 
    SELECT * FROM employees

    • Выбираем только уникальные значения температуры в таблице с измерениями

Используемые команды:

  • SELECT - Выбрать что-то, откуда-то
  • FROM name - Из таблицы/базы данных с именем name
