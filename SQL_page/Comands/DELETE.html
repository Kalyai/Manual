DELETE в SQL

Оператор DELETE в языке SQL используется для удаления одной или нескольких записей из таблицы базы данных. Также он позволяет установить условия для конкретики удаления.

Примеры:
  • Удаление без условия 
    DELETE * FROM employees

    • Удаляем все записи в таблицу `employees`

  • Удаление с условием 
    DELETE * FROM employees
WHERE age > 50

    • Удаляем только те столбцы, где возраст сотрудника > 50

Используемые команды:

  • DELETE - Удалить записи
  • FROM name - Из таблицы/базы данных с именем name
  • WHERE - Оператор условия
