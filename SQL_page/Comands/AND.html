AND в SQL

Оператор AND в языке SQL используется для объединения нескольких условий в запросе. Это логическое И в мире информатики.

Примеры:
  • В связке с WHERE 
    SELECT * FROM employees
WHERE salary > 60000 AND age < 30

    • В первой строчке мы делаем запрос на выбор все строчек из таблицы `employees`

    Во второй строчке мы создаем условия для выбора строчек: (ЗП > 60000) И (возраст < 30)

  • В синтаксисе BETWEEN 
    SELECT * FROM products
WHERE price BETWEEN 10 AND 50

    Первая строчка такая же, только таблица с другим именем

    Дальше добавляем условия: Цена от 10 до 50 включительно

Используемые команды:

  • SELECT - Выбрать столбцы из таблицы/базы данных (знак `*` означает выбрать все)
  • FROM name - Из таблицы/базы данных с именем name
  • WHERE - Оператор для добавления условий
  • BETWEEN - Для обозначения диапазона
  • AND - Логичекое `И`, нужна для условий
