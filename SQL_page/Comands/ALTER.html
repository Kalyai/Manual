ALTER в SQL

Команда ALTER в SQL используется для изменения структуры существующей таблицы в базе данных. С её помощью можно добавлять, удалять или изменять разные части таблицы, такие как столбцы, ключи, ограничения.

Примеры:
  • Добавление/удаление стобца 
    ALTER TABLE employees
ADD COLUMN birthdate DATE;

// Также при добавление/удаления CONSTRAINT - ограничений

ALTER TABLE employees
DROP COLUMN birthdate;

    • Мы сначала добавили в таблицу `employees` колонку `birthdate` с типом данных `DATE`

    Потом удалили колонку `birthdate` в таблице `employees`

  • Изменение типа данных столбца 
    ALTER TABLE employees
ALTER COLUMN salary SET DATA TYPE DECIMAL(10, 2)

    Мы изменили тип данных в таблице `employees` в колонке `salary` на `DECIMAL`

Используемые команды:

  • ALTER - команда для внесений изменений
  • TABLE name - таблица с именем name
  • ADD - добавление чего-либо
  • COLUMN name - колонка с именем name
  • DROP - удаление чего-либо
  • SET DATA TYPE - установить тип данных: ...
