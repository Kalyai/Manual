INDEX в SQL

INDEX в SQL - это специальная структура данных, которая позволяет ускорить операции поиска и выборки в таблицах базы данных. Индексы работают подобно указателям, которые указывают на местоположение данных в таблице, что позволяет базе данных быстро находить нужные строки без необходимости перебора всех записей.

Примеры:
  • Создание индекса для одного столбца 
    CREATE INDEX fast_search ON employees(name)

    • Создаем индекс `fast_search` (можно любое имя) для столбца `name` в таблице `employees`

  • Индекс, который предотвращает дублирование значений 
    CREATE UNIQUE INDEX any_name ON employees(name)

    • Создаем ункальный индекс для столбца `name` в таблице `employees`

  • Создание индекса для двух и более столбцов 
    CREATE INDEX need_memory ON employees(name, age, ...)

    • Создаем индекс `need_memory` для столбцов `name`, `age` и любых других в таблице `employees`

Используемые команды:

  • CREATE - Создание объекста в указанном месте
  • INDEX - Ускоряет работу со строками, занимает больше памяти
  • ON - Создает связь, указывает куда что-то добавить, сделать и т.п. (указатель)
