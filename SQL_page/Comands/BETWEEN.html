BETWEEN в SQL

Оператор BETWEEN в языке SQL используется для фильтрации данных в пределах заданного диапазона значений. Он позволяет выбирать строки, в которых значение столбца находится между двумя указанными значениями, включая сами эти значения.

Примеры:
  • Сотрудники с возрастом между 18 и 20 
    SELECT * FROM employees
WHERE Age BETWEEN 18 and 20

    • Выводим всю инфомарцию из таблицы `employees`

    Где числовое значение строки с возрастом между 18 и 20 включительно

Используемые команды:

  • SELECT - Выбрать столбцы из таблицы (знак `*` значит все столбцы)
  • FROM name - Из таблицы/базы данных с именем name
  • WHERE - Оператор для добавления условий
  • BETWEEN - Проверка числового значения на диапазон между number1 AND number2
